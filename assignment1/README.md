##  問題說明與設計理念 


本次作業要求有以下功能，對於影像要有以下幾點操作:

1. ROI裁切
2. 影像旋轉
3. 影像縮放
4. 結果輸出

##  主要程式架構說明 

### 文件架構

```
|--assignment1 #作業要求以及原文件
        |--Assignment1.pdf
        |--*.jpg
|-src #程式存放
        |--utils
                |--modules
        |--main.py
|-modifypic #存放使用者修改後的圖片
        |--*.jpg
|-README.md
```

### 程式結構

為了方便維護，在開發過程中盡量遵循 SOLID 開發原則

透過 python 的 opencv package `cv2` 進行影像操作

最後透過視窗呈現與原本影像的對比，並需要儲存更改後的圖片
